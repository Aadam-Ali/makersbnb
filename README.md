MVP User Stories:

```
As a user,
So that I can start a session,
I would like to be able to enter my name
```

```
As a user,
So that I can choose a space,
I would like to see a list of all spaces
```

```
As a user,
So that I can book a space,
I would like to view a space's details and confirm my booking
```

```
As a user,
So that I know my booking was placed,
I would like to see a confirmation screen
```
